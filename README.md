The Goal of this project was 2 main point 



1: make a movement system, basic enough to get from A to B and enough to facilitate a "special move" that i like.

2: make a character, and i have to like the result





I also wanted to see how long does something like this take...





There was some sub goal:



Sub goal 1: a movement system, A to B,  need a A and a B, a testing ground of sort.

Sub goal 2: make it look good and package it into a short experience, 3 A to B fetch Quest, few short cutscene, a ending choice, a small narrative, personal narrative and world narrative.
